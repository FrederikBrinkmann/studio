# **App Name**: RAUM5 Digital

## Core Features:

- Navigation: Sticky navigation bar with links to About, Services, Cases, Approach, and Contact sections. Includes a burger menu for mobile.
- Hero Section: Display a strong headline and subline with two CTA buttons: 'Projekt anfragen' and 'Intro-Call buchen'. An abstract data visualization is optional.
- Services Cards: Showcase Data Science & Analytics, Softwareentwicklung, and KI Consulting services using cards with titles, short explanations, and bullet points. Provides subtle hover effects.
- Team Grid: Display team member profiles (Martin Johannes Brucker, Frederik Brinkmann) with round images, focus badges, and subtle hover animations.
- Case Studies: Showcase fictional customer projects such as 'Digitale Prozessplattform im Gesundheitswesen', 'KI-gestützte Dokumentenanalyse im Finanzbereich', 'Predictive Analytics für operative Planung'.
- Approach Timeline: Presents the four-step approach: Verstehen, Konzipieren, Umsetzen, Skalieren.
- Contact Form: A form with fields for Name, Company, Email, Project Description, and a checkbox for a 30-min Intro-Call, Submit-Button with client-side validation.

## Style Guidelines:

- Background color: Dark grey (#0A0A0A) for a modern, minimalist look.
- Primary text color: White (#FFFFFF) for clear readability.
- Secondary text color: Light grey (#D3D3D3) for softer contrast.
- Accent color: Soft blue (#ADD8E6) for buttons and highlights.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern feel.
- Note: currently only Google Fonts are supported.
- Use of generous whitespace to maintain a clean and airy feel, inspired by high-end portfolio designs.
- Subtle fade-in animations and hover effects on buttons to enhance user experience without being overwhelming.
